# 示例脚本

此目录包含量化交易Skill套件的示例脚本。

## 示例列表

| 脚本 | 说明 |
|------|------|
| 01_quick_start.py | 快速入门示例，演示如何获取数据并进行简单分析 |
| 02_full_workflow.py | 完整工作流程示例，演示如何使用多个Skill |

## 运行示例

```bash
# 快速入门
python examples/01_quick_start.py

# 完整工作流程
python examples/02_full_workflow.py
```

## 注意事项

示例脚本使用了模拟数据，如需真实数据，请配置相应的数据源（如BaoStock、Tushare等）。

更多详细内容请参考项目根目录下的：
- [QuickStart.md](../QuickStart.md) - 5分钟快速入门
- [Example_Workflow.md](../Example_Workflow.md) - 完整量化工作流程示例
