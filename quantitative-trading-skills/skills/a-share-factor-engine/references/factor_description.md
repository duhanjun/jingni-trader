# 因子说明

## 技术因子

### momentum_1m (1个月动量)
- **定义**: 过去20个交易日的涨跌幅
- **理论依据**: 动量效应，过去表现好的股票未来继续表现好
- **预期IC方向**: 正

### momentum_3m (3个月动量)
- **定义**: 过去60个交易日的涨跌幅
- **理论依据**: 中长期动量效应
- **预期IC方向**: 正

### reversal_1m (1个月反转)
- **定义**: 过去20个交易日涨跌幅的相反数
- **理论依据**: 短期反转效应，A股市场常见
- **预期IC方向**: 正（因为是反转，因子值越大预期收益越高）

### volatility_20d (20日波动率)
- **定义**: 过去20个交易日日收益率的标准差
- **理论依据**: 低波动异象，低波动股票有更高风险调整收益
- **预期IC方向**: 负

### rsi_14 (14日RSI)
- **定义**: 相对强弱指标
- **理论依据**: 超买超卖信号
- **预期IC方向**: 视市场环境而定

### macd (MACD)
- **定义**: MACD指标（DIF线）
- **理论依据**: 趋势跟踪
- **预期IC方向**: 视市场环境而定

### bollinger (布林带位置)
- **定义**: (收盘价 - 20日均线) / 20日标准差
- **理论依据**: 均值回归
- **预期IC方向**: 负（超卖反弹）

### kdj_k, kdj_d (KDJ的K和D)
- **定义**: KDJ指标的K值和D值
- **理论依据**: 超买超卖
- **预期IC方向**: 视市场环境而定

### willr (威廉指标)
- **定义**: 威廉指标
- **理论依据**: 超买超卖
- **预期IC方向**: 视市场环境而定

## 基本面因子（框架预留）

### lncap (对数市值)
- **定义**: 市值的对数
- **理论依据**: 小盘股溢价
- **预期IC方向**: 负

### turnover (换手率)
- **定义**: 换手率或成交量
- **理论依据**: 流动性溢价
- **预期IC方向**: 负

## 使用提示

1. **因子预处理**: 建议先对因子进行缩尾（Winsorize）处理，减少极值影响
2. **标准化**: 推荐对因子进行截面标准化（z-score），使不同因子可比
3. **行业中性化**: 若需要，可对因子进行行业中性化处理
