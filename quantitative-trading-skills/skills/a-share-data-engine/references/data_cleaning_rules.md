# A股数据清洗规则

## 清洗流程

### 1. 缺失数据处理

- 将数值类型的字段转换为 float
- 删除 code 或 date 为空的记录
- 对于可选字段，允许空值存在

### 2. 停牌标记

- 如果成交量为 0，则标记为停牌
- `is_suspended = (volume == 0)`

### 3. 涨跌停标记

- 涨跌幅在 [9.8%, 10.2%] 之间为涨停
- 涨跌幅在 [-10.2%, -9.8%] 之间为跌停
- ST股票涨跌幅限制为 5%（可配置）

### 4. ST股票过滤

- 股票名称中包含 "ST" 或 "*ST" 标记为 ST
- 可配置 `FILTER_ST=True` 过滤掉 ST 股票

### 5. 新股过滤

- 计算上市天数 = 交易日期 - 上市日期
- 可配置 `FILTER_NEW_STOCK_DAYS` 过滤新股
- 默认过滤上市不足 60 天的股票

### 6. 缺失日期填充

- 按股票代码分组
- 生成完整日期范围
- 用前值或 NaN 填充缺失日期

## 复权处理

复权处理由各个数据源适配器实现：

- **Tushare**: `adj="qfq"` 或 `adj="hfq"`
- **BaoStock**: `adjustflag="3"` (前复权)
- **AkShare**: `adjust="qfq"`

复权因子计算：
- 前复权 = 原始价格 × 复权因子
- 后复权 = 原始价格 ÷ 复权因子

## 数据一致性检查

- 检查 open ≤ high, low ≤ high, close 介于 low 和 high 之间
- 检查 volume ≥ 0, amount ≥ 0
- 检查涨跌幅计算是否正确
